Your Todo List

Todoapp is an application where you can create your own list of tasks which you wanna do or you've already done and just want to mark it as done. You can see the information about every task that is in process. You can see how many days this task is in process or set desired time for your task. You can edit your task anytime or delete it if you want. Start your todo list and create your tasks so you can see the time you need to do the task.

A few steps to start your todo list:

- Sign up or login if you're already registered

- Open your tasks list and create your task

- Click on the information button and then edit button if you want to change the status or information {% if user.is_authenticated %}
